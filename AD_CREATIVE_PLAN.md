## Ad Placement Strategy

### Platform Specifications

**Instagram/Facebook Feed**:
- Video: 1:1 (square) or 4:5 (vertical)
- Image: 1:1 or 4:5
- Duration: 15-60 seconds
- Use concepts: 1, 2, 3, 5
- Images: 1, 2, 4

**Instagram/Facebook Stories**:
- Video: 9:16 (vertical full screen)
- Image: 9:16
- Duration: 15 seconds max
- Use concepts: 2, 6
- Images: 3, 5

**TikTok**:
- Video: 9:16 (vertical)
- Duration: 15-60 seconds
- Use concepts: 1, 2, 4, 5
- Favor UGC style (concept 4)

**YouTube Pre-roll**:
- Video: 16:9 (landscape)
- Duration: 15-30 seconds (skippable at 5s)
- Use concepts: 1, 5, 6
- Hook must be in first 5 seconds

**Google Display Network**:
- Image: 1200x628 (landscape), 300x600 (vertical), 300x250 (square)
- Use images: 2, 4, 6

## Production Workflow

### Video Editing Checklist
- [ ] Export all source videos from assets folder
- [ ] Create project folder structure (by concept)
- [ ] Edit in 9:16 (TikTok/Stories) first, then crop for 1:1, 4:5, 16:9
- [ ] Add text overlays using bold sans-serif font (Helvetica/Arial Black)
- [ ] Color grade for consistency (maintain Bloom brand colors)
- [ ] Add music (royalty-free or licensed tracks)
- [ ] Include 1-2 second pause before CTA for comprehension
- [ ] Export formats: MP4 H.264, 30fps minimum
- [ ] Size limits: <4MB for Stories, <1GB for Feed

### Static Image Design Checklist
- [ ] Use app store screenshots as base assets
- [ ] Maintain Bloom brand colors (extract from app icon)
- [ ] Typography: Bold headlines, clean body copy
- [ ] High contrast for text overlays (readability on mobile)
- [ ] Test designs at mobile size before finalizing
- [ ] Export as PNG or JPEG (max quality)
- [ ] Create multiple aspect ratios per design
- [ ] A/B test variations (different headlines, CTAs)

## A/B Testing Strategy

### Variables to Test

**Headlines**:
- "Stop using ChatGPT for stocks" vs "ChatGPT vs Bloom AI"
- "Build a portfolio in 12 seconds" vs "AI builds your portfolio instantly"
- "Invest smarter" vs "Invest like a pro"

**CTAs**:
- "Download free" vs "Try Bloom AI" vs "Start investing smarter"
- "Link in bio" vs "Get the app" vs "Download now"

**Hooks** (first 3 seconds):
- Pattern interrupt ("WAIT STOP!") vs Question ("What if you invested...?") vs Statement ("I lost $8,400...")
- Comparison (side-by-side) vs Testimonial (UGC) vs Feature showcase

**Duration**:
- Short (15s) vs Medium (30s) vs Long (60s)
- Test by platform and audience

**Creative Style**:
- Polished/professional (3D rotato) vs Raw/authentic (UGC) vs Educational (feature carousel)

## Creative Best Practices

### Mobile-First Design
- Readable text at small sizes (minimum 24pt font)
- Bold colors and high contrast
- Captions/subtitles for sound-off viewing
- CTA visible within first 3 seconds

### Platform-Specific Optimization
- **TikTok**: Native feel, trending audio, UGC style
- **Instagram**: Polished, aspirational, visually cohesive
- **Facebook**: Clear value prop, social proof, longer form OK
- **YouTube**: Story-driven, educational, skippable hook

### App Install Best Practices
- Show actual app interface (not just marketing graphics)
- Demonstrate core value prop in <10 seconds
- Clear CTA with app icon
- Feature faces when possible (UGC, testimonials)
- Use motion to grab attention in feed
