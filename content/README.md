## Common Elements

All storyboards follow the proven viral formula:

### Structure
- **Frame 1 (0-5s)**: Controversial hook
- **Frame 2 (5-30s)**: Context and setup
- **Frame 3 (30-60s)**: Evidence and breakdown
- **Frame 4 (60-90s)**: Call-to-action

### Key Patterns
✅ Specific names and numbers (Nancy Pelosi, $20M, 45 days)
✅ Political/financial controversy
✅ "Us vs. them" framing
✅ Data-driven storytelling
✅ Strong CTAs with audience participation

### Visual Style
- Clean, minimal graphics
- Dark mode color scheme
- Bold typography
- Animated charts and timelines
- Real screenshots when possible

### Audio
- Direct, conversational voiceover
- Minimal suspenseful background music
- Fast-paced editing (2-3 second cuts)

## Usage

Each storyboard is production-ready and includes:
- Frame-by-frame breakdown
- Visual descriptions
- Voiceover scripts
- Text overlay suggestions
- Timing recommendations
